# 🎬 Random GIF Generator

A fun and interactive Random GIF Generator built with **React** and the **GIPHY API**. Generate random GIFs and explore GIFs based on your interests with a simple and responsive interface.

## ✨ Features

* 🎲 Generate random GIFs
* 🔍 Search for GIFs by keywords
* ⚡ Fetch GIFs using the GIPHY API
* ⏳ Loading state while fetching GIFs
* ❌ Error handling for failed API requests
* 📱 Responsive user interface


## 🛠️ Tech Stack

* **React.js**
* **JavaScript**
* **GIPHY API**
* **CSS / Tailwind CSS**
* **Vite**
* **Axios / Fetch API**

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone YOUR_REPOSITORY_URL
```

### 2. Navigate to the project

```bash
cd random-gif-generator
```

### 3. Install dependencies

```bash
npm install
```

### 4. Add your API key

Create a `.env` file in the root directory:

```env
VITE_API_KEY="your_giphy_api_key"
```

> Make sure `.env` is included in your `.gitignore` file.

### 5. Start the development server

```bash
npm run dev
```

Open the local development URL shown in your terminal.

## 🔑 API

This project uses the **GIPHY API** to fetch GIF content.

You will need your own GIPHY API key to run the project locally.
